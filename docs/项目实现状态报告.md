```yaml
      expertise: ["专业领域", "前沿发展", "技术趋势"]

```

#### 3.2 评审流程设计
1. **文档类型识别**：自动识别文档类型（简历、论文、公文、合同）
2. **目标分析**：提取文档目标信息（求职岗位、投稿期刊、公文类型、合同性质）
3. **角色选择**：根据文档类型和目标智能选择评审角色
4. **并行评审**：多个角色同时进行专业评审
5. **结果整合**：汇总各角色评审意见，生成综合建议
6. **优先级排序**：按重要性和紧急程度排序建议
